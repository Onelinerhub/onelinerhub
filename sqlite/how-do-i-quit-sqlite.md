# How do I quit SQLite?
// plain

To quit SQLite, type `.exit` in the console.

## Example

```
sqlite> .exit
```

This will exit the SQLite console and return to the terminal.

## Code explanation

- `.exit`: This is the command used to quit the SQLite console.
