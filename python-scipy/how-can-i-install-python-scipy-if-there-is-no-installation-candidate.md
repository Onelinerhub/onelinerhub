# How can I install Python-Scipy if there is no installation candidate?
// plain

**Installing Python-Scipy without an installation candidate**

1. Download the source code for the latest version of Python-Scipy from the [Python-Scipy GitHub repository](https://github.com/scipy/scipy).
2. Unzip the source code and navigate to the unzipped directory.
3. Install the necessary dependencies for Python-Scipy by running the command `pip install -r requirements.txt`
4. Compile the source code by running the command `python setup.py build`
5. Install the compiled code by running the command `python setup.py install`
6. Verify that the installation was successful by running the command `python -c "import scipy; scipy.test()"`
7. If the tests pass, you have successfully installed Python-Scipy without an installation candidate.
