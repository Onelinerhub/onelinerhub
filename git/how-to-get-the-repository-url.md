# How to get the repository (remote) url

```bash
git remote -v
```

- `git` - base command
- `remote -v` - will list all remotes

group: remote

## Example: 
```bash
git remote -v
```
```
origin	git@github.com:Onelinerhub/onelinerhub-ui.git (fetch)
origin	git@github.com:Onelinerhub/onelinerhub-ui.git (push)
```
