# How to create Numpy matrix

```python
import numpy as np
m = np.matrix([[1, 2], [3, 4]])
```

- `import numpy as np` - load [lib:Numpy module](/python-numpy/how-to-install-python-numpy-lib) for Python
- `[[1, 2], [3, 4]]` - sample matrix data
- `np.matrix` - creates matrix from given data

## Example: 
```python
import numpy as np
m = np.matrix([[1, 2], [3, 4]])
print(m)
```
