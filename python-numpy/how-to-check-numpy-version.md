# How to check Numpy version

```python
import numpy as np
ver = np.__version__
```

- `import numpy as np` - load [lib:Numpy module](/python-numpy/how-to-install-python-numpy-lib) for Python
- `np.__version__` - attribute with Numpy version

group: quickstart

## Example: 
```python
import numpy as np
print(np.__version__)
```

1.22.3
