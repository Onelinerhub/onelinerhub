# How do I install and use Sphinxsearch with Homebrew?
// plain

1. Install Sphinxsearch with Homebrew:
   ```
   brew install sphinx
   ```

2. Start Sphinxsearch:
   ```
   sphinx start
   ```

3. Stop Sphinxsearch:
   ```
   sphinx stop
   ```

4. Configure Sphinxsearch:
   ```
   sphinx conf
   ```

5. Index data with Sphinxsearch:
   ```
   sphinx indexer --all
   ```

6. Search data with Sphinxsearch:
   ```
   sphinx search <query>
   ```
