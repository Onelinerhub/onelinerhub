# Convert number to string

```lua
str = tostring(num)
```

- `str` - string variable that will contain result
- `tostring` - converts given value to string
- `num` - number to convert to string

group: convert_str

## Example: 
```lua
str = tostring(12.3)
print(type(str))
print(str)
```
```
string
12.3

```
