# How to view object attributes

```python
attributes = dir(object)
```

- `attributes` - list of `object` attributes
- `dir(` - will return list of all attributes of specified object
- `object` - object to list attributes of

## Example: 
```python
object = 'hi'
attrs = dir(object)
print(attrs)
```
