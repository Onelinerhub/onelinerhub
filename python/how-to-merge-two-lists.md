# How to merge two lists

```python
merged_list = [1, 2, 3] + [4, 5, 6]
```

- `merged_list` - will contain resulting list merged from two
- `[1, 2, 3]` - first example list
- `[4, 5, 6]` - second example list

group: merge_objects

## Example: 
```python
merged_list = [1, 2, 3] + [4, 5, 6]
print(merged_list)
```
```
[1, 2, 3, 4, 5, 6]

```
