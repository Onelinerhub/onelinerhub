# How to add an element to the list

```python
list = [1, 2]
list.append(5)
```

- `list` - example list
- `append` - will append specified element to the end of specified list
- `5` - element to add to our list

## Example: 
```python
list = [1, 2]
list.append(5)
print(list)
```
```
[1, 2, 5]
```
