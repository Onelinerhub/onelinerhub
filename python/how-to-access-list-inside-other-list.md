# How to access list inside other list

```python
list = [[1, 2, 3], [3, 4, 5]]
print(list[1][2])
```

- `list` - example list with nested lists
- `list[1][2]` - will access third element (index `2`) of second nested list (index `1`)

## Example: 
```python
list = [[1, 2, 3], [3, 4, 5]]
print(list[1][2])
```
